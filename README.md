# StrawFramework

## 一般约定

* url 参数使用下划线划分割
* 变量名, 方法名, 类名, 文件名, 驼峰法命名, 文件/文件夹名/ Namespace 首字母必须大写, 其他首字母小写
* 数据表字段名使用下划线分割
* 输出字段使用下划线分割
